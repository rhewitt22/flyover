This experiment uses Mapbox-GL to create 3D flyovers of roads and trails on lands managed by the U.S. Fish and Wildlife Service, National Wildlife Refuge System.

The refuges chosen for this demo come from [Great Drives on National Wildlife Refuges](https://www.fws.gov/story/great-drives-national-wildlife-refuges).

## Config

This app requires a `config.js` file in the root of the src folder with an API key from Mapbox.

```const MAPBOX_ACCESS_TOKEN = 'pk....';```

## Development

This app was built using Node v22.22.0.

### Setup

To run this project, clone the repository:

```git clone https://github.com/rhewitt22/flyover.git flyover```

Install the dependencies:

```npm install```

Run the development server:

```npm run dev```
