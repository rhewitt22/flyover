# Project Name
Refuge flyover

## Description
This web application will allow visitors to experience national wildlife refuges from afar. The application allows users to fly over important areas on a refuge to understand their size and layout.

## Features
- Stores data in a file that is easy to edit
- Uses mapbox GL for smooth flyover animations.
- Uses free and open imagery with the best resolution.
- User should be able to provide multiple flyover paths via data file
- UI should allow user to change between different flyover paths
- Each path should have a play button with fast forward and rewind controls and progress bar

## Configuration
There should be a data folder that contains a CSV file for providing flyover data. Please include an example of how to fill in each field.
